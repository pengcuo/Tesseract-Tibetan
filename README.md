# Tesseract-Tibetan
## OpenCV && Tesseract-OCR in Android Studio
藏文软件紧缺，而且少的很可怜。有些仅仅有着藏文科技公司的名字，不实惠藏族同胞。身为一名藏族，服务藏民是义不容辞的责任。在忙里偷闲当中在GitHub上找到这个Tesseract-OCR，重要的是支持藏文，废话不多说开始了。
